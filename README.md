# Online Retail Data Cleaning Project

## Objective
Clean and prepare the Online Retail dataset for analysis.

## Dataset
Online Retail Dataset from UCI Machine Learning Repository.

## Tools Used
- Python
- Pandas
- Google Colab
- GitHub

## Data Cleaning Steps
- Removed duplicate rows
- Removed missing CustomerID and Description values
- Removed cancelled orders
- Removed invalid Quantity and UnitPrice values
- Converted InvoiceDate to datetime format
- Created a new TotalAmount column

## Files Included
- Online_Retail_Cleaned.csv
- data_cleaning.ipynb
- data_dictionary.xlsx

## Outcome
Prepared a clean, analysis-ready dataset for future business insights.
